# 💫 星轨盯盘 - Windows 小白使用说明书

欢迎使用 **星轨量化盯盘系统**！这是一个完全运行在您本地电脑上的量化盯盘与选股系统，不需要配置任何复杂的 Python 运行环境。

以下是专为电脑小白编写的 **Windows 一键开箱即用启动指南**：

---

## 🛠️ 第一步：安装运行环境 (只需做一次)

系统运行需要一个名为 **Node.js** 的软件（它是运行本网页本地服务器的引擎）。

1. **下载安装包**：
   - 打开电脑浏览器，访问官网：[https://nodejs.org/zh-cn](https://nodejs.org/zh-cn)
   - 点击页面上的 **`LTS (长期支持版)`** 绿色按钮进行下载。
2. **开始安装**：
   - 双击下载好的安装文件（例如 `node-v20.xx.x-x64.msi`）。
   - 在弹出的安装向导中，一路点击 **`Next` (下一步)** 和 **`Install` (安装)**。
   - 期间如果弹出 Windows 安全警告（是否允许此应用修改您的设备），请选择 **`是`**。
   - 安装完成后，点击 **`Finish` (完成)**，并**重启一次您的电脑**。

---

## 📂 第二步：解压代码包

1. 找到您收到的代码压缩包（例如 `stock-monitor.zip`）。
2. **右键点击压缩包**，选择 **`全部解压缩...`**，将文件解压到一个硬盘目录，例如：`D:\stock-monitor`。
   - *注意：请勿直接在压缩包内双击启动，务必先进行解压缩！*

---

## 🚀 第三步：一键双击启动

1. 打开您刚才解压出来的文件夹。
2. 找到名为 **`双击启动盯盘.bat`** 的文件，直接双击它。
3. **系统将自动执行以下操作**：
   - **首次运行**：会自动在窗口中为您安装所需的网页依赖组件（只需 1~2 分钟，请耐心等待它自动完成）。
   - **自动开机**：自动启动本地量化盯盘网页服务器，并由系统在就绪后自动在您的浏览器中打开盯盘界面（通常为 `http://localhost:5173/` 或 `http://localhost:5174/`，脚本会自动为您适配空闲的端口）。
4. **以后使用时**，依然只需双击运行该 `.bat` 文件，网页就会瞬间自动启动并弹出。

---

## 📈 第四步：使用与数据更新

1. **更新K线行情数据**：
   - 首次打开网页时，请点击导航栏右上角的 **`更新K线`** 按钮。
   - 系统会自动下载全市场最新的 A 股日K行情，并保存到您本地。下次打开时会**瞬间秒级载入**，无需再次下载。
   - 您只需在交易日收盘后（下午3点后），点击一次 `更新K线` 即可自动增量下载最新的行情。
2. **筛选个股**：
   - 点击左侧的“扫描全市场”按钮，即可根据均线与突破参数筛选出符合条件的个股。
3. **自选股管理**：
   - 在右侧“筛选结果”列表下的个股卡片底部，点击 `加入自选`。
   - 切换右侧顶部的标签到 **`自选股`**，您就可以实时点击个股查看其 K 线图，并随时通过底部的红色 `移出自选` 按钮来移除它们。
   - 自选股列表会自动安全地保存在项目目录下的 `src/data/favorites.json` 文件中，即使清空浏览器缓存也不会丢失！
